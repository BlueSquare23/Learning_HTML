Displaying an Image

Use the "img" tag to link to an image in an html document.
For Example,

<img src="URL" alt="image description" >

The "img" tag takes two attributes, "src" & "alt"

* "src" stands for source and this is the location of the image.

* "alt" stands for Alternate Text and it is often used as a simple description of the image. This can be useful for people who use sceen readers or if for some reason the image is not able to be rendered.

Image Display Size

By default images will display at their actual size.

We can modify this with the use of the "width" and "hight" attributes.
For example,

<img scr="URL/Path" alt="description" width="200px" height="230px">

Note: If you only scale one of these atributes then it will automatically scale the other one while keeping the same aspect ratio.

Using Images

Images can be placed dynamically within a line of text as well.
For example,

<p>Text here<img scr="Path" alt="Description" width="50px">more text!</a></p>
