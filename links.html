Simple link

The simplest link is created using the <a> tag like this,

<a href="location">link</a>

For example,

Visit johnlradford.me

The above example is what is known as a global link meaning that is the path to the front page of that site no matter where you click that link from. However there is also another kind of link known as a,

Relative Link

Relative links are named as such because they follow relative paths. An inportant thing to keep in mind about relative links is that they can only be use from a particular location. Whereas a global link can be put in any page anywhere a relative link has to reference a page which lives in the same directory structure as itself.

Here is an example of a relative link.

* Source,

<a href="./more_formatting.html">Relative link</a> to a different page from our current location.

* Rendered,

Relative link to a different page from our current location.

Linking Within Pages

As well as linking to pages, we may also link to specific parts within a page. We may do this by either creating a named anchor or placing an id attribute on a tag that is at the place on the page we would like the link to take us.

To link to different section have either an id or anchor tag placed in that section. Think simply link to that tag.

For example,

Jump to section 2

It is generally advised to use id's rather than names.

Link Titles

Link titles give make a little title pop up whenever a user hovers over a link. Link titles can be created with the use of title elememnt.

* Source,

<a href="https://johnlradford.me" title="A Pretty Cool Website!">johnlradford.me</a>

* Rendered,

johnlradford.me
